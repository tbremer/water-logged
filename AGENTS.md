# AGENTS.md

Guidance for AI coding agents working in this repository (**Water Logged**, a
standalone watchOS hydration app built with SwiftUI + SwiftData/CloudKit).

## RootView changes

When modifying `WaterLogged/Views/RootView.swift`, **comment out the existing
code — do not delete it.**

- Never remove the canonical `RootView` body (the `TabView` containing
  `TodayView` / `HistoryView` / `SettingsView`).
- If you need `RootView` to render something else temporarily — for example,
  showing a single screen like `SettingsView()` for a screenshot or preview —
  **comment out** the `TabView` and add the temporary view, then restore it by
  uncommenting.
- This keeps the canonical navigation from being lost and makes any temporary
  override trivially reversible.

## Project notes

- The Xcode project is generated by XcodeGen from `project.yml`;
  `WaterLogged.xcodeproj` is git-ignored. Run `xcodegen generate` after you
  add, rename, or move files (not needed for edits to existing files).
- `xcodegen generate` only rewrites `WaterLogged.xcodeproj` — it never touches
  your source files.
